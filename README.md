# 로또 번호 추첨기

한국 로또(6/45) 규칙에 맞는 웹 번호 추첨기입니다. UI는 `DESIGN1.md` 가이드(Bugatti 스타일)를 따릅니다.

## 실행 방법

### 방법 1 — 바로 열기 (추천)

`lotto.html` 파일을 더블클릭하거나 브라우저로 드래그하면 바로 사용할 수 있습니다.

### 방법 2 — 개발 서버

Node.js가 설치되어 있다면:

```bash
npm install
npm run dev
```

브라우저에서 표시되는 주소(기본: http://localhost:5173)로 접속하세요.

## 기능

- 1~45 중 중복 없는 6개 번호 + 보너스 번호 1개 추첨
- 한 번에 최대 5세트까지 추첨
- 실제 로또와 동일한 번호대별 색상 표시
- 추첨기 애니메이션으로 공을 하나씩 꺼내는 연출
- 역대 1등 당첨번호 조회 (1회~최신 회차, 1등 당첨금·당첨자 수 포함, 회차·번호 검색)
